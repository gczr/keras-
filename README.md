### 准备工具

Python2:

- Flask==1.0.2
- gevent==1.4.0

Python3:

- tensorflow==1.13.1
- Keras==2.2.4
- pandas==0.23.4
- numpy==1.16.2
- scikit-learn==0.21.3

### 运行步骤：

1. 切换至hualos文件夹，运行api.py脚本，命令为:

```bash
python api.py
```

2. 在浏览器中打开网址： http://localhost:9000

3. 运行脚本model_train.py，命令为:

```
python3 model_train.py
```
